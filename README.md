## 📌 Overview

This project demonstrates hands-on packet capture and network traffic analysis using Wireshark in a simulated Security Operations Center (SOC) environment.

It focuses on identifying unencrypted traffic, reconstructing communication sessions, and analyzing protocols to detect potential security risks.

## 🔍 Lab Included

### 🧪 Capturing a Packet Using Wireshark

- Packet capture from a live network interface  
- HTTP traffic filtering and inspection  
- TCP and HTTP stream reconstruction  
- Protocol analysis for security insights
